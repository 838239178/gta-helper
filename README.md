# gta-helper
gtaV 名钻 dc 破解
## 部署  
python 3.6+ (本人py37)  
- pip install -r requirements.txt  
启动gta-finger.py
## 配置文件 config.json  
  ```json
 {"resolution":"1920x1080", //全屏化分辨率
            "screenmode":"window", //window/fullscreen 无边框窗口化/全屏
            "keycooldown":0.01, //按键抬起之间的冷却(默认10ms)觉得自己电脑可以的可以调成0
            "pointcooldown":2.5 //点点点两次选位置之间的冷却
}
```
